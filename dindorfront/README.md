# Dindorfront - Application Web Dind'or

Application web professionnelle pour la société Dind'or - Gestion d'élevage de volaille.

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 19.2.19.

## Pages disponibles

L'application comprend trois pages principales :

- **Welcome** (`/welcome`) : Page d'accueil avec présentation de l'application et fonctionnalités
- **Login** (`/login`) : Page de connexion avec authentification
- **Signin** (`/signin`) : Page d'inscription avec formulaire complet

## Design

Le design utilise une palette de couleurs professionnelle inspirée du thème de la volaille :
- Rouge foncé (#8B1A1A) pour la couleur primaire
- Doré (#DAA520) pour la couleur secondaire
- Design moderne avec animations et transitions fluides
- Responsive design pour tous les appareils

## Development server

To start a local development server, run:

```bash

```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.
